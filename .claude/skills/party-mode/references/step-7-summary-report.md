# Step 7 — Summary Report to User

After the Root Agent's review returns `accepted`, the Root Agent reports back to the user.

The report must include:

- What was done (feature implemented / bug fixed)
- Files changed (taken from the sub-agent results and the review)
- Tests added or updated (taken from the tester result)
- The review outcome and any non-blocking recommendations left for follow-up
- Any outstanding notes or follow-up recommendations

Also update the `Status` column in the `## Task List` section of the markdown plan document.
